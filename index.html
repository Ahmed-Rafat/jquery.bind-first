jQuery.bind-first

bind-first is a tiny jQuery plugin which allows you to attach a handler to an event for the elements (like regular $.fn.on, $.fn.bind, $.fn.delegate and $.fn.live do) and put it on the first place in the event queue.

By using jQuery.bind-first you may be sure that your event handler will be executed first, regardless of its actual appearance in the code.

Usage

$(/* selector */).onFirst(/* arguments  */);
$(/* selector */).bindFirst(/* arguments  */);
$(/* selector */).delegateFirst(/* arguments  */);
$(/* selector */).liveFirst(/* arguments  */);

The arguments are completely identical to the corresponding jQuery functions: on, bind, delegate and live.

Example

$('#bind a')
    // regular bind: binded first, will be runned second
    .bind('click', function() {
        alert('regular bind');
        return false;
    })
    // bindFirst: binded second, will be runned first
    .bindFirst('click', function() {
        alert('bindFirst');
        return false;
    });
